# PawHub - Project Context

## Project Name
PawHub

## Product Summary
PawHub is a premium hybrid pet marketplace for dogs and cats only in India.

The platform combines:

1. Peer-to-peer users who want to rehome or put pets up for adoption.
2. Verified breeders/sellers who can list pets professionally.
3. Buyers/adopters who browse and connect safely.

## Main Goal
Build India's most trusted and modern pet marketplace with premium UI, strong trust systems, and excellent mobile UX.

---

## Core User Roles

### User
- Register / Login
- Browse pets
- Save favorites
- Contact seller
- Rehome pet
- Adopt pet

### Verified Seller
- Sell dogs/cats
- Manage listings
- Receive verified badge

### Admin
- Approve listings
- Approve seller verification
- Remove spam/fake listings
- View analytics
